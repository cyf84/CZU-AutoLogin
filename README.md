# 常州工学院 - 校园网自动连接工具 (CZU AutoLogin)

这是一个基于 Python 编写的轻量级 Windows 桌面端工具。它可以静默运行在后台，在开机或夜间断网恢复后，自动帮你登录常州工学院的校园网。

## ✨ 核心特性
- **纯粹静默：** 后台运行，几乎零资源占用。
- **开机自启：** 智能写入系统启动项，一次配置，永久无感连网。
- **断网重连：** 完美应对 11:40 的夜间断网规则。
- **图形配置：** 首次运行提供友好的 UI 界面进行配置。

## 🚀 快速使用 (面向普通同学)
1. 在 [Releases] 页面下载最新版的 `.exe` 程序。
2. 双击运行，在弹出的窗口中填入纯学号、选择运营商并输入密码。
3. 点击保存，即可享受无感冲浪。

## 💻 开发者指南
如果你想自己编译或修改代码：
1. 克隆本项目：`git clone ...`
2. 安装依赖：`pip install requests`
3. 运行：`python czu_autologin.py`
4. 打包 EXE：`pyinstaller -F -w czu_autologin.py`
